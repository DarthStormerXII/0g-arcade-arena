# 0G Arcade Arena Pitch Deck

Date: 2026-06-25

## Brand System

- Direction: fully purple liquid amethyst chrome on near-black glass, lifted directly from `public/brand/logo.jpg`, `public/brand/thumbnail.jpg`, and `public/brand/hero.mp4`.
- Palette: void black `#08020d`, deep violet `#10051c`, amethyst `#b56cff`, soft violet `#c084fc`, chrome lavender `#e7c7ff`, pale lilac `#f3e8ff`.
- Typography: Sora for title, UI, and deck text. Heavy uppercase headings, short proof labels, no generic neon-green terminal framing.
- Motif: reflective purple glass ribbons, orb crop, and thin lavender proof glints. Use the thumbnail as full-bleed title/section-divider art.
- Avoid: cyan, gold, green, yellow, or generic terminal-neon accents. Proof states can still be named `green` or `yellow` in code, but visible deck colors must resolve to purple/lavender.

## Slide Story

1. Title: 0G Arcade Arena
   - Visual: full-bleed purple chrome thumbnail with the logo orb as the center mark.
   - Message: The game lobby for ownable agents.

2. Problem: Agent games are fragmented
   - Visual: violet glass strips separating one-off demos, isolated agents, and hidden results.
   - Message: Builders lack shared rooms, replay receipts, agent reputation, and proof surfaces.

3. Product: Browse, play, prove, submit
   - Visual: four-step purple liquid ribbon with game cards embedded in the chrome surface.
   - Message: A multi-game arena where humans and agents compete under one proof standard.

4. Live Demo: Four v1 games
   - Visual: screenshots or clips from Grid Four, Fleet Duel, Tile Race, and World Cup Draft.
   - Message: The submission is not a single mini-game; it is a reusable arcade protocol.

5. Agent Competition
   - Visual: agent profile chips on purple glass with wager/free match states rendered in lavender, amethyst, and deep violet only.
   - Message: Qualified agents can enter supported games, play legally, and build a result trail.

6. Builder Platform
   - Visual: PR-only game pack checklist, manifest hashes, and validator commands on dark violet panels.
   - Message: New games enter through reviewable game packs, not untrusted uploads.

7. 0G Proof Layer
   - Visual: explorer receipts, Storage roots, Galileo chain commits, and DA candidate labels as lavender proof cards.
   - Message: Every proof claim is either live-proven on 0G or truthfully labeled fallback/pending.

8. Why It Wins
   - Visual: chrome orb over three callouts: playable, extensible, verifiable.
   - Message: Arcade Arena gives 0G a public place where agent gameplay can compound.
